Projetos

Minima Studio (Landing Page)

Minima Studio é uma landing page pessoal criada para uma empresa fictícia de design visual. O projeto foca em branding, interfaces digitais e experiência moderna, com navegação fluida, design responsivo e elementos minimalistas. Ideal para portfólios ou apresentação de negócios criativos.

Deploy GitHub

DevLoop (Plataforma de Mentoria)

DevLoop é uma plataforma web completa para mentoria de desenvolvedores. Permite cadastro de mentores e mentorados, agendamento de sessões, gerenciamento de disponibilidade, chamadas de vídeo (WebRTC), chat integrado, avaliações e upload de vídeos. Backend em Java Spring Boot e frontend em React, com foco em comunicação, acompanhamento profissional e experiência responsiva.

Deploy GitHub

Luna (Gestão Financeira)

Luna é uma aplicação web para gerenciamento de dinheiro, criada com TypeScript e Vite. Permite controlar gastos, receitas e visualizar o saldo de forma intuitiva e responsiva. Ideal para quem busca organização financeira pessoal ou empresarial, com foco em usabilidade e design moderno.

Deploy GitHub

Lista de Tarefas Moderna

Aplicação web desenvolvida como desafio técnico, focada em acessibilidade, usabilidade e visual moderno. Permite adicionar, editar, excluir e filtrar tarefas por status e tags, definir data de vencimento, alternar entre temas claro/escuro e ajustar tamanho da fonte. Totalmente responsiva, com persistência local e navegação por teclado.

Deploy GitHub

Orbitr (Kanban de Projetos)

Orbitr é uma aplicação web de Kanban para gestão visual de tarefas e colaboração em equipes. Possui backend em Java Spring Boot e frontend em Angular, com recursos como quadros, tarefas com comentários, anexos, etiquetas, notificações por e-mail, autenticação JWT e integração com PostgreSQL. Ideal para organizar projetos, atribuir tarefas e acompanhar o progresso em tempo real.
